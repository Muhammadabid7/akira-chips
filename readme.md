```

## 🚀 Getting Started

1. **Clone the repository**
   ```sh
   git clone https://github.com/your-username/akira-potato.git
   cd akira-potato
   ```

2. **Install dependencies**
   ```sh
   pnpm install
   ```

3. **Run the development server**
   ```sh
   pnpm dev
   ```

4. **Open in your browser**
   ```
   http://localhost:3000
   ```
